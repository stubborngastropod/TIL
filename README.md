### CLI(Command Line Interface)

**GUI와 CLI**

CLI는 GUI에 비해 보기는 더 불편하지만 적은 성능을 요구한다.

**명령어(mkdir, touch, rm, cd, ls, start/open)**

- mkdir: 폴더 생성
- touch: 파일 생성
- rm: 파일 삭제(rm -r: 폴더 삭제)
- cd: 디렉토리 변경
- ls: 현재 작업 위치
- start: 파일 실행

**절대경로/상대경로**

../를 통해 상위 폴더로 이동
